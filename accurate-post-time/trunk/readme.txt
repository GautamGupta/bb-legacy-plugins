=== Accurate Post Time ===
Contributors: fel64
Requires at least: 0.8.3

Outputs an accurate post time in, optionally, your choice of formatting. Use <?php aposttime( 'DATE FORMAT' ); ?> in your template, <a href="http://uk3.php.net/date">date formatted according to PHP standards.</a>

== Description ==

Outputs an accurate post time in, optionally, your choice of formatting. Use <?php aposttime( 'DATE FORMAT' ); ?> in your template, <a href="http://uk3.php.net/date">date formatted according to PHP standards.</a>

== Installation ==

* Add the `accurate-post-time` file to bbPress 'bb-plugins/' directory and activate.
* For example, in front-page.php, change the "topic_time()" to "topic_time('l j M G:i')
* Enjoy :)
