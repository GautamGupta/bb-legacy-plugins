Allows the Administrators to add users through the administration panel.

== Description ==

This plugin adds a new function called `Add User` to the administration. There the keymaster is able to add new users to the forum and notify them by email.

== Installation ==

Add `admin-add-user.php` to your `/my-plugins/` directory.

== Configuration ==

not needed.

== Frequently Asked Questions ==

= I'm using bbpress & Wordpress together? =

So then, you don't really need this plugin, because Wordpress already has this feature!
