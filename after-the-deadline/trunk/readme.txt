After the Deadline Plugin checks spelling, style, and grammar in your bbPress forum posts.

== Description ==

After the Deadline helps you write better and spend less time editing. Activate the Plugin and click the link above the post textbox to check spelling, style, and grammar.

NOTE - Only English language is supported for the time being.

This plugin is based on After the Deadline by:
Mudge, Raphael S. (2009) After the Deadline – Language Checking Technology.
[Automattic](http://open.afterthedeadline.com)

== Installation ==

1. Upload the extracted `after-the-deadline` folder to the `/my-plugins/` directory
2. Activate the plugin through the 'Plugins' menu in bbPress
3. Enjoy and always write correct English!
