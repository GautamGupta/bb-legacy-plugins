Shows Gravatar and identicons.

== Description ==

Allows you to define your own default Gravatar. You can also choose to use identicons (wavatar, identicon or monsterid). You can also change the size of the avatars

== Installation ==

Add `bb-avatars.php` to your `/my-plugins/` directory.


== Configuration ==

Activate on your pluginspage, and change the settings in the admin section as you wish.
