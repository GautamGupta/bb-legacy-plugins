== Description ==

BB-to-WP Users Copier (Aka bbwpuc) copies users from BBpress forum to Wordpress users database, useful when integrating BBpress and Wordpress. 

This is a beta release for review. Any feedback, bug reports, improvements.. you are welcome.

== Installation ==

1. install plugin in it's own directory in `my-plugins` then activate plugin 
2. adjust settings under `Settings -> BB-to-WP users copier`
