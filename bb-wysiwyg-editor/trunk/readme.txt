== Description ==

Add Wysiwig Editor features and allows quoting of posts when replying.

== Installation ==

1. If you had a previous installation of BB Wysiwig Editor, remove your theme customizations for this plugin. They're not needed anymore.
1. Upload the folder `bb-wysiwyg-editor` to your `/my-plugins/` directory.
1. Activate the plugin.
1. You're done :)

== Configuration ==

None necessary.
