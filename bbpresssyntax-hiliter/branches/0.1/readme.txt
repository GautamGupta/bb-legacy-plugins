Syntax Highlighter for various programming languages using GeSHi.

== Description ==
	
With this plugin, users may include highlight code in their posts.
see http://klr20mg.com/bbpress_syntax_highlighter-en/ for more info

== Installation ==

Add the `bbpress_syntax_hilite` folder to bbPress' `my-plugins/` directory.
