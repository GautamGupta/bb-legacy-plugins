== Description ==

Blocks posts via your own list of words or IP addresses (like the WordPress feature) by immediately marking them as spam.

== Installation ==

* Copy the `blocklist/` directory and files to your plugins directory and activate. 

* Blocklist settings and instructions can be found under the Manage admin menu.

== Frequently Asked Questions ==

= Why mark them as spam? =

* bbPress unfortunately does not currently support any other status or queue types so it's the easiest way to do this

= Where did the Manage admin menu come from? =

* The Manage menu is now added to bbPress 1.x automatically by some of my plugins for backwards compatibility with bbPress 0.9
