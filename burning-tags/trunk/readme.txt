Display only tags from topics that were posted to this month in the tag cloud.

== Description ==
When hot tags aren't hot enough, burning tags will do the trick!

Only topics that have been posted on in the last 30 days show up in the list, and topics with more posts are more important on a logarithmic scale.

The tag cloud on the tags page is not affected by this plugin.

== Installation ==
1. Upload the entire `burning-topics` folder to your `my-plugins` folder.
2. Activate the plugin in the plugins admin panel.

== Frequently Asked Questions ==
= What exactly is the equation used to find the importance of a tag? =
`Importance = Topics + ln( Posts )`
