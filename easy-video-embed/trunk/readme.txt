=== Easy Video Embed ===
Tags: video embed, myvideo, vimeo, flashvideos, flashvideo, youtube, videos, video, media, film, multimedia, embedded, embedding, embed
Requires at least: 1.0
Tested up to: 1.02
Stable tag: 0.1

This plugin allow bbPress forums administrators and members to easily embed Videos in posts, by using bbCode video tag.

== Description ==

Using Easy Video Embed, it is so easy to embed videos inside posts, for admins and forum members. All you have to is to
use the [video] bbCode tag to put the URL of the video source inside the [video][/video] tags, is that SIMPLE!

We currently supports the following Tube video sources:
YouTube, Glumbert. Google Video. Metacafe. MyVideo.at. Yahoo! Videos, Myspace,
LiveLeak.,Revver, Vimeo, Guba, GameVideos, GameTrailers, Tudou (Single/Playlist),
CollegeHumor, 5min.

== Installation ==

1. Upload the extracted `easy-video-embed` folder to the `/my-plugins/` directory
2. Activate the plugin through the 'Plugins' menu in bbPress
3. Enjoy!

== Frequently Asked Questions ==

= 1. What is Easy Video Embed? =
You may read more about Easy video Embed Plugin on its [about page](http://www.bbpressplugin.com/easy-video-embed-plugin/).

= 2. How do I change the size of the embedded videos? =
Well, it is pretty simple. all you have to do is to click the 'Easy Video Embed' menu link, and in the
options settings you will see input fields where you can put any size (width & height) for any supported video source.
