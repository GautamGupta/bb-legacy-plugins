Auto-delete users who have not done first-time login within specific hours.

== Description ==

Please visit this [plugin's website](http://trac-hg.assembla.com/llbbsc/wiki/bbER) for more information, submitting bugs, or discussing [here](http://www.livibetter.com/it/forum/llbb-small-creations).

= Features =

  * Automatic deletion
  * Report of deletion
  * Complete deactivation 

== Installation ==

  * Unpack, then put directory `EnhancedRegistration` to `my-plugins`.
  * Activate, then set time limit.
  * Add a notice for new users, remind them doing first-time login within time limit.
