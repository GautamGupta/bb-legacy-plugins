A simple plugin for forum redirection, when a user clicks on a forum they will be redirected to another link.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `forum_redirector.php` to the `bbpress\bb-plugins\` directory
2. Activate the plugin through the 'Plugins' menu in BBPress
3. Edit the plugin as indicated within the .php file
