== Description ==

Shows icon next to topic title to indicate hotness or popularity based on number of posts and topic age.

== Installation ==

* Add the `hot-topic` directory to bbPress' `my-plugins/` directory and activate.

== Frequently Asked Questions ==

*  I intend to make this more sophisticated eventually and actually look at number of recent posts in the past hours/days by number of users
