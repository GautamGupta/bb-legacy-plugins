== Description ==

Allows the administrator to specify titles for users based on their
current post count.

The titles appear in next to the users posts.

== Installation ==

Unzip the file and copy the post-count-titles.php file into the my-plugins
directory.

If the my-plugins directory is not present you need to create it under
the root directory of your forum.

The plugin then needs to be activated and configured in the admin area.
