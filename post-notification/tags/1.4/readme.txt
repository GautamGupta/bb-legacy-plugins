Sends an notification email if there is a new post on user's favorites.

== Description ==

It does what the name says. If you have a topic in your favorites and there will be a new post to this topic it sends you an email. Simply upload and it does its job.

== Installation ==

Add `notification.php` to your `/my-plugins/` directory.

== Configuration ==

not needed.

== Frequently Asked Questions ==

= How can I enable it? =

Every user can enable or disable this feature in his profile settings.
