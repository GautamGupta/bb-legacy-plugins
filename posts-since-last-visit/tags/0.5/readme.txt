Displays all new posts since user's last visit.

== Description ==

This plugin displays all new posts since user's last visit as view. This is still beta, fell free to try it out. Requires Simple Onlinelist 1.4!

== Installation ==

Add `since-last-visit.php` to your `/my-plugins/` directory.

== Configuration ==

not needed.
