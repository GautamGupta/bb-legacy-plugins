Shows keymaster/administrators the most recent posts across all forums via the admin menu

== Description ==

With this plugin keymaster/administrators may view the most recent posts 
and use (extremely limited) moderation options under the admin menu.

It's very similar to the "edit comments" built into wordpress.

== Installation ==

Add the `recent-posts.php` file to bbPress' `my-plugins/` directory.
Activate and check under "content" admin submenu for "Recent Posts".
