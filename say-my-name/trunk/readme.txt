== Description ==

This plugin will notify you if your nick will be called inside a post or topic.

Thanks to Thomas Klaiber for his plugin that allowed me to build this plugin.

== Installation ==

Simply download the file, and upload to /my-plugins dir.

Then you have only to activate the plugin by your personal profile.
