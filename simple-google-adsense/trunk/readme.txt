== Description ==

Add Google Adsense code in your bbPress forum, in the footer :) ALL USERS CAN BE IMPROVED IT :P

== Installation ==

* Add the `SimpleGoogleAdsense.php` file to bbPress' `bb-plugins/` directory.

* Activate and check under "settings" admin submenu for "Google Adsense Configuration".
