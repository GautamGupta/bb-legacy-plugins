Allows bbPress to send email through an SMTP server.

== Description ==

This plugin allows an administrator to direct emails sent by bbPress through
an SMTP email server.

This plugin only works with bbPress version 1.0-alpha or higher.

== Installation ==

1. If you don't have a /my-plugins/ directory in your bbpress installaltion, 
   create it on the same level as bb-config.php.

2. Upload the plugin file smtp-mailer-for-bbpress.php into /my-plugins/ directory

Open the file in a text editor and configure the settings as necessary.
