== Description ==

Turns any number of forums into support forums, where users can mark topics as resolved, not resolved or not a support question.

It is possible to add icons to support topics, as well as to closed and sticky ones. A default set of 6 icons (1 per status + 1 "not resolved" alternative icon) is included.

A view for each of the support statuses may also be generated.

Based of <a href="http://www.adityanaik.com/">so1o</a>, <a href="http://blogwaffe.com/">mdawaffe</a> and <a href="http://unlettered.org/">SamBauers</a>' <a href="http://bbpress.org/plugins/topic/support-forum/">Support Forum</a> plugin.

<strong>bbPress 1.0 fully compatible!</strong>

== Installation ==

* Copy plugin folder into `my-plugins` folder.

* Activate the plugin and go to plugin configuration page at Plugins > Support Forums.

* Select the forums you want to turn into support forums and take a look at the other available options.
