Description: Display posts as threads

== Description ==

Adds a Reply link below each post so that the post page can show posts in a threaded manner.

Current Limitations:

* No pagination of posts
* CSS needs some love

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Unzip the plugin into the `my-plugins` directory
1. Activate the plugin through the 'Plugins' menu
