Share user accounts between your WordPress blog and your bbPress forums.

== Description ==

This plugin ensures that pre-existing WordPress users are able to login to your bbPress forums.

== Installation ==

Add the `wordpress-integration.php` file to bbPress' `my-plugins/` directory.
