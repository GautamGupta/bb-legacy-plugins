Adds a css file to bbPress to make the administration page the same colors as WordPress.

== Description ==

WP-CSS adds a css file to bbPress to make the administration page the same blue colors as WordPress. That's pretty much it.

== Installation ==

1. Upload `wp-css.php` and the folder `wp-css` and its contents to your plugins directory (`my-plugins`).
2. Activate the plugin through the 'Plugins' menu in bbPress
