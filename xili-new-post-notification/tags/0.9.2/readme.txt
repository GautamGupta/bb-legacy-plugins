A very light plugin to send a short notification to a choosen email address when a new post is created.

== Description ==

Simple plugin to send short notification to a choosen email address when a new post is created. Useful for simple forum with only one moderator.

== Installation ==

* Add the `xili-bb-npn.php` file (or his folder) to bbPress' `my-plugins/` directory and activate.
* Go to Settings sidebar menu and New Post Notification submenu. Fill the to: address and save settings...
