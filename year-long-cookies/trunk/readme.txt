Force login cookies to last a year instead of the default two days (or two weeks). Works with 0.8.x, 0.9.x and 1.0a

== Description ==

* Forces login cookies to last a year instead of the default two days (or two weeks). 

== Installation ==

* This is an auto-load plugin (and must have a leading underscore `_`). Put into your plugin directory and you're done, no activation required.

== Frequently Asked Questions ==

* This plugin name must retain the leading underscore for auto-load  ie. `_year-long-cookies.php`
