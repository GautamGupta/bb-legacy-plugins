== Description ==

* Force login cookies to a year instead of the default two days (or two weeks). Works with both the old 0.8.x password method and the new method in the trunk.

== Installation ==

* This is an auto-load plugin (and must have a leading underscore `_`). Put into your plugin directory and you're done, no activation required.

== Frequently Asked Questions ==

* This plugin name must retain the leading underscore for auto-load  ie. `_year-long-cookies.php`
