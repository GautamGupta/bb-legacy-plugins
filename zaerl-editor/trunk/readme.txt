== Description ==

<span style=\"font-family: Georgia, \'Times New Roman\', Times, serif; font-size: 16px;\"> <p style=\"margin-top: 0px; margin-right: 0px; margin-bottom: 1em; margin-left: 0px; line-height: 21px; padding: 0px;\">zaerl Editor adds a little toolbar with useful buttons on top of the editor area. Each of these buttons automatically adds the html code of one of the allowed markup. No template edit required. You can specify a custom CSS for the toolbar as well as the CSS selector used to find the textarea.<p style=\"margin-top: 0px; margin-right: 0px; margin-bottom: 1em; margin-left: 0px; line-height: 21px; padding: 0px;\">jQuery is used in order to generate the HTML code needed and the code is loaded only on those template page that actually have a post form. Italian and spanish translations are included.</p> </span></p>

== Installation ==

Extract the zip file and just drop the contents in the my-plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
