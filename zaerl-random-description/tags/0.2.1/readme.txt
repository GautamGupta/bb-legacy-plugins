Generate a random site description from a custom list of sentences.

== Description ==

zaerl Random Description generate a random site description from a list of sentences that can be specified through an admin panel. The site description is updated every time a new page is loaded. 
  
Italian translation is included.

== Installation ==

Extract the zip file and just drop the contents in the my-plugins/ directory of your bbPress installation and then activate the Plugin from Plugins page.
