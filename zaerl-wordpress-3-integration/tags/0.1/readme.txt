Wordpress 3 deep integration fix.

== Description ==

This plugin fix a problem that occurs during user registration when bbPress is deep integrated with WordPress 3.

== Installation ==

Extract the zip file and just drop the contents in the my-plugins/ directory of your bbPress installation and then activate the Plugin from Plugins page.
